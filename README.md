# BAQIYA

BAQIYA est un outil de comptoir pour gérer le manque de petite monnaie.

❌ Pas une banque  
❌ Pas un wallet  
❌ Pas un paiement  

✅ Tickets de monnaie  
✅ Offline-first  
✅ Usage terrain

## Stack
- Fastify
- Prisma
- PostgreSQL
- React PWA

## Règle absolue
Si une fonctionnalité ressemble à une fintech, elle est refusée.
